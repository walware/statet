Navigation

Beside the common actions to move the cursor and select ranges in text editors in Eclipse, the R editor offers additional content aware ways to navigate inside the R editor and the Eclipse workspace.

Hyperlinks

Hyperlinks allows to access context sensitive actions using the mouse. Hyperlinks are enabled on demand by pressing a modifier key or key combination; if the hyperlink action is available at the mouse position, the link is shown in the editor and can be triggered by a mouse click.

There are several hyperlink actions for the R editor including actions to navigate:

Name Description / Action Default Modifier(s)
Element Declaration Opens the declaration of an R element in Eclipse
(available for detected R element names), like the command Open Declaration		 Ctrl
Local File Path Opens the file in an editor in Eclipse
(available for detected path names in string)		 Ctrl

Configuration

The different kind of hyperlinks can be configured in the preferences (Window > Preferences) at: General > Editors > Text Editors > Hyperlinking Only items listed as Available In "R Source" in the table are relevant for R code.
