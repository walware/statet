Console View

The Eclipse Console View displays a variety of console types including the R Console. The Console view can manage multiple console represented by their pages. Only one page can be shown at once by the view. To see multiple consoles at the same time, it is possible to open multiple views.

Common Actions of the Console View

Name Description Availability
Pin Console

If one pins a console by toggling the Pin Console button, it means that other consoles that might be opened (R Consoles or SVN consoles, TeXlipse consoles, etc.) will not open inside the current Console View, but inside a separate, new Console View. The pinning of an R console can also be set when configuring an R Launch Configuration.

 View toolbar
Console Button

If several consoles are opened in Eclipse, the console button allows to navigate to the pages of the other consoles that are available within the Console View. When two consoles are open, a single click navigates between the two consoles.

 View toolbar
Next Page

If several consoles are opened in Eclipse, this command switches to the next console in the list of open consoles.

 Local command
Alt+F7
Previous Page

If several consoles are opened in Eclipse, this command switches to the previous console in the list of open consoles.

 Local command
Alt+Shift+F7
Open Console

This drop down menu offers several options to open a new Console view or other consoles. When opening additional Console views it is possible to show multiple R consoles side by side. If the SVN support Subclipse is installed, the menu allows to open the SVN Console.
