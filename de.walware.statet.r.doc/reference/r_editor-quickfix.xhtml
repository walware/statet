Quick Fix

Use of Quick Fix Assist

Quick Fix is a context-sensitive mechanism in Eclipse that helps the user in quickly fixing problems or performing a task. For the moment, Quick Fix in the R editor supports renaming R variable names across a file. To invoke the Quick Fix assistant use Ctrl+1.

As can be seen on the screenshot, the user can rename variables globally in the file, or choose to change only the given occurrence and all preceding (respectively following) occurrences of the variable.
